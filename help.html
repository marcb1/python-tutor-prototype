Website overview!

This describes the current features that have been implemented in our design.
For future features that have not been implemented please check our powerpoint.

The sidebar

The sidebar displays all the current lessons that are available for you to use. If you want a very short overview of each lesson (before deciding which lesson you want to read), click the all lessons link in the sidebar. Currently the only lesson we have implemented, is the hello world lesson. Every lesson has an interactive tutor at the bottom of the page so you can start coding and receive feedback in real time.

The breadcrumbs

The breadcrumbs are under the website title so you don't get lost while going through our website. If you want to get back to where you were or figure out how you got to a certain page, check the breadcrumbs.

The navigation bar

The navigation bar has a search bar so you can search for certain lessons that you are looking for. It also has a link to the interactive tutor if you just want to program without having to go to a lesson page.

The interactive tutor

Jump right into coding, our interactive tutor will guide you along and suggest code improvements in real time while you type!
