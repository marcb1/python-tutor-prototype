The print function

Python comes with some built in functions. the print function, prints the string given to it and a new line unlike the C printf function.
We call the python print function like so: print "hi". This tutorial uses python2, in python3 they tried to make the syntax similar to a regular function call. In python3 we call print like so: print("hi)

python3

print("hello world!")

python2

print "hello world!"

Indentation

function print_hello_world:
  print "hello world!"
  print "spaces must be consistent!"
Instead of curly braces python uses indentation, the standard says 4 spaces. But as long as you are consistent the interpreter doesn't care. The above is a function that prints hello world, notice how the indentation is consistent.
